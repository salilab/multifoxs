<li><a href="https://salilab.org/foxs/"> FoXS </a> - profile calculation server

<li><a href="https://salilab.org/foxsdock/"> FoXSDock </a> - protein-protein docking with SAXS profiles

<li><a href="http://bl1231.als.lbl.gov/bilbomd">BILBOMD</a> - BILBOMD allows you to determine the three-dimensional domain structure of proteins based on conformational sampling using a molecular dynamics (MD) approach. Conformational sampling is followed by structure validation using FoXS and MultiFoXS.

<li><a href="https://integrativemodeling.org/"> IMP </a> -
Integrative Modeling Platform for comprehensive structural
characterization of biomolecules. MultiFoXS code is found in the kinematics, foxs, and multi_state modules.

<li><a href="http://www.cgl.ucsf.edu/chimera/"> UCSF Chimera </a> - can be used to fill in missing fragments in your input protein.
FoXS is also available through Chimera interactive visualization program. See
this <a href="http://www.cgl.ucsf.edu/chimera/docs/ContributedSoftware/saxs/saxs.html">
tutorial</a> on how to use FoXS from Chimera

<li><a href="http://www.bioisis.net"> Bioisis</a> - open access SAXS database, SAXS tutorials

<li><a href="http://www.embl-hamburg.de/ExternalInfo/Research/Sax/software.html"> EMBL SAXS software </a>

<li><a href="http://www.sasbdb.org/"> SASBDB </a> - Curated repository for small angle scattering data and models
